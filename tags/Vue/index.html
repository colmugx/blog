Object.defineProperty() 方法会直接在一个对象上定义一个新属性，或者修改一个对象的现有属性， 并返回这个对象。 —— MDN

Proxy 对象用于定义基本操作的自定义行为（如属性查找，赋值，枚举，函数调用等）。 —— MDN
